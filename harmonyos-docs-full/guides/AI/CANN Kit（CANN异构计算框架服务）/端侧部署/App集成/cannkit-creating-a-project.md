# 创建项目

 

本章以[Caffe SqueezeNet](https://github.com/forresti/SqueezeNet)模型集成为例，说明App集成操作过程。

 

1. 创建DevEco Studio项目，选择“Native C++”模板，点击“Next”。

 

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/6c/v3/JzGiHT_iQjuJU97yaUOu1g/zh-cn_image_0000002573975151.png?HW-CC-KV=V1&HW-CC-Date=20260420T191330Z&HW-CC-Expire=86400&HW-CC-Sign=B177E542D4685AA3DE0038E2AFDC21AD4D384F1FC3D748342B5A573C64559D6D)
2. 按需填写“Project name”、“Save location”和“Module name”，选择“Compile SDK”为“5.1.0(18)”及以上版本，点击“Finish”。
