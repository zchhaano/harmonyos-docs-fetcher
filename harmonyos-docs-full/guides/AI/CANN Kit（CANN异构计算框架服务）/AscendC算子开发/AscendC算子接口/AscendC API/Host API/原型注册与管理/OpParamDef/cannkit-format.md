# Format

  

#### 函数功能

定义算子参数数据格式。

  

#### 函数原型

```
OpParamDef &Format(std::vector<ge::Format> formats);

```

  

#### 参数说明

 

| 参数 | 输入/输出 | 说明 |
| --- | --- | --- |
| formats | 输入 | 算子参数数据格式，ge::Format请参考 Format 。 |

   

#### 返回值

[OpParamDef](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cannkit-paramtype)算子定义。

无
