## 函数原型

```
OpParamDef & DataType ( std :: vector < ge :: DataType > types );
```

 展开

| 参数 | 输入/输出 | 说明 |
| --- | --- | --- |
| types | 输入 | 算子参数数据类型，ge::DataType请参考 DataType 。 |

## 返回值

[OpParamDef](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cannkit-opparamdef)算子定义。
