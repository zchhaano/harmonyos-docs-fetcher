## 函数功能

注册的具体网络类型的自动映射函数。

## 函数原型

收起自动换行深色代码主题复制

```
void AddAutoMappingSubgraphIOIndexFunc (domi::FrameworkType framework, AutoMappingSubgraphIOIndexFunc fun) ;
```

## 参数说明

 展开

| 参数 | 输入/输出 | 说明 |
| --- | --- | --- |
| framework | 输入 | 网络类型，FrameworkType类型定义请参考 FrameworkType 。 |
| fun | 输入 | 自动映射输入输出函数，函数类型详见 AutoMappingSubgraphIndex 。 |

## 返回值

无

## 异常处理

无

## 约束说明

无
