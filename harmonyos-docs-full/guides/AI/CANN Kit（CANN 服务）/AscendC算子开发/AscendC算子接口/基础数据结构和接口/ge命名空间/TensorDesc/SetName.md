## 函数功能

向TensorDesc中设置Tensor的名称。

## 函数原型

注意

数据类型为string的接口后续版本会废弃，建议使用数据类型为非string的接口。

  收起自动换行深色代码主题复制

```
void SetName ( const std::string &name) ; void SetName ( const char_t *name) ;
```
