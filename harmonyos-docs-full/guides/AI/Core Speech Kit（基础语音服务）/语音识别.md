# 语音识别

将一段中文音频信息（中文、中文语境下的英文；短语音模式不超过60s，长语音模式不超过8h）转换为文本，音频信息可以为PCM音频文件或者实时语音。

## 场景介绍

手机/平板等设备在无网状态下，为听障人士或不方便收听音频场景提供音频转文本能力。

## 约束与限制

  展开

| AI能力 | 约束 |
| --- | --- |
| 语音识别 | 支持的语种类型：中文普通话。 支持的模型类型：离线。 语音时长：短语音模式不超过60s，长语音模式不超过8h。 |
