# 移动和缩放模拟器

- 移动模拟器

您可以使用鼠标拖动模拟器到屏幕的指定位置。首先将鼠标放在屏幕边缘，当鼠标变成![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226102021.08425856579125566699694590435976:50001231000000:2800:815AC14C5103D852B8BBECE904B91D94BFEE7896C137D4352A6DB87026DE64FF.png)样式，按住鼠标左键并移动即可拖动模拟器。当模拟器被拖动到期望位置后，松开鼠标左键即可停止拖动。
- 缩放模拟器

如需改变模拟器大小，将鼠标放到屏幕四角的任意一处，当鼠标变成![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226102021.06198605325177422929804280177556:50001231000000:2800:04CCB98B69D2E3AF6686CA77B0BC2C05379CA5677E530581898386F8355A5CAC.png)，按住鼠标左键并移动即可缩放模拟器。当模拟器被缩放到期望大小后，松开鼠标左键即可完成缩放。
