# 启动和关闭模拟器

在设备管理器页面，单击![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226102018.96875884533945074137897220975015:50001231000000:2800:FAADF6A80C855F5E6AFCAB7C482AE8F6E74671C045F534EDCED31BFF5F47D0A3.png)即可启动模拟器。模拟器启动时会默认携带上一次运行时的用户数据，包括用户上传的文件，安装的应用等。如果是新创建的模拟器，则不会携带用户数据。如果想清除上一次运行时的用户数据，点击**Actions >**![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226102018.43761828876332826565631003521573:50001231000000:2800:E69DCA589716CF08EC9029BB91928F38F227FAEBD7A1F1FEE5A6A70CCCB094B3.png)**> Wipe User Data**。

![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226102019.15141224247066466903216565286137:50001231000000:2800:5E177EE5EE6F3CCF2B88A46CC55E895E3D3509BB196AFFC6E1A519BEF8975363.png)

在模拟器运行期间，可以点击**Actions >**![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226102019.22533653683706523542057701368552:50001231000000:2800:A87BD6AC227500F605E378639EEE8F31DB787C3754D1801730AF3DB34110DFEE.png)**> Show on Disk**显示模拟器在本地生成的用户数据。点击**Actions >**![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226102019.58030080087035802246558115984439:50001231000000:2800:CF5931491E1CA13FDE06F9D5B208154B36F55937253225F3534F7CA5C519128B.png)**> Generate logs**可以生成模拟器自启动到此刻的所有日志信息。想要关闭运行时的模拟器，可以在设备管理器页面点击![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226102019.86424988645438295319864972943355:50001231000000:2800:069A0EBDAFE59F321E9C7A6CB42C7E6D5CE49B6EA0A8C6B2FAF528D882775671.png)，或者点击模拟器工具栏上的关闭按钮![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226102019.88370074045883199875198269300885:50001231000000:2800:09685DE2318BCA9561CDE51162BCC37EF60266B2F7A8C6AA4B266199E0E67704.png)。

![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226102019.30557961359996690115630405449605:50001231000000:2800:3144814BEAAAF00E0B3215C71B2D42F4F50B03CBB35D23AC9AE5AE47C98A7CEC.png)

模拟器关闭后，点击**Actions >**![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226102019.32409319548333049685933730129475:50001231000000:2800:ECA738747C6C95AB1DD18C8ECB6A98438E2B37AE02FDCA6535CB13B30E1678E4.png)**> Delete**可以删除模拟器，并清除模拟器的用户数据和配置信息。
