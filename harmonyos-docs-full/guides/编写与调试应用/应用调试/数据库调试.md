# 数据库调试

 

从DevEco Studio 6.0.2 Beta1版本开始，新增Database Inspector，支持在DevEco Studio上执行SQL语句查看、修改应用数据库，无需将应用数据库先导出到本地，提升开发调试效率，当前支持SQLite和向量数据库。

 

开发者也可以通过命令行工具调试数据库，具体操作方式请参考[vector-store数据库调试工具指导](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/vector-store-debug-tool)、[SQLite调试工具指导](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/sqlite-database-debug-tool)。

 

#### 使用场景

- 应用某个操作会修改数据库内容，可以实时查看修改后的内容是否符合预期。
- 可以实时修改数据库内数据，构造测试场景，提升测试效率。

  

#### 使用约束

- 设备系统需要不低于API 22。低于API 22时，SQLite数据库仅支持查看，不支持修改，向量数据库无法使用。
- 不支持使用release签名的应用。
- 仅支持调试database目录下的数据库。
- 不支持调试加密数据库，建议使用非加密库。
- 不支持调试隐私用户、多用户数据库，建议使用默认用户。
- 不支持导入导出数据库。
- 执行SQL时，存在以下约束：

  - 不支持执行多条SQL。
  - 存在多条SQL时，不支持高亮选择后执行单条SQL。
  - SQL执行不支持历史记录。
  - SQL执行不支持事务。

  

#### 操作步骤

1. 点击菜单栏**View > Tool Windows > Database Inspector**，打开Database Inspector。

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/bd/v3/OGuVliv7RqmjbwWmsTk1_w/zh-cn_image_0000002561752909.png?HW-CC-KV=V1&HW-CC-Date=20260420T194020Z&HW-CC-Expire=86400&HW-CC-Sign=C33F005B5F35F1B358FD18B4E3770F13B860029F4F3CD35DF1A8109FBF3287BB)

 

Database Inspector打开后，页面各区域作用如下：①选择设备，②选择应用包名，③数据库和表信息展示，④SQL执行和数据查看。

 

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/6e/v3/-6o5IWXjRZOQCf_RZ15thQ/zh-cn_image_0000002561832887.png?HW-CC-KV=V1&HW-CC-Date=20260420T194020Z&HW-CC-Expire=86400&HW-CC-Sign=794F5D4B1B4D0276DF2385D6AC4A1780F444AA5585820C03864711A7F140CF24)
2. 从设备下拉列表中选择设备（设备需已连接）。
3. 选择包名，点击右侧的**Connect to Databases**按钮，即可查看数据库相关信息。（如果使用DevEco Studio 6.0.2 Beta1版本，按钮名称是**Execute**）。

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/1d/v3/OvthPAABS7O1uy-NH3tL5w/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260420T194020Z&HW-CC-Expire=86400&HW-CC-Sign=1AD05011548AC4B7B67D48A39CA482B3E905ECD14BE8F5D8C70E42359613E1ED) 

设备系统版本低于API 22时，Database Inspector会将数据库下载到本地计算机，界面上显示的是本地计算机路径，设备系统为API 22及以上版本时，界面上显示的是设备上的数据库路径。

  

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/4e/v3/vvsP1CmgTZKQDbWAe6qA7g/zh-cn_image_0000002530752962.png?HW-CC-KV=V1&HW-CC-Date=20260420T194020Z&HW-CC-Expire=86400&HW-CC-Sign=297BAABDD8273238CCE17E9B3DAB27B89D7949C4D908922E78E15F63570FAA4E)
4. 双击数据库表名，右侧区域展示表数据，默认按照20条/页展示。

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/2b/v3/mUey7wV8QkqD-e89h4nwOA/zh-cn_image_0000002561752907.png?HW-CC-KV=V1&HW-CC-Date=20260420T194020Z&HW-CC-Expire=86400&HW-CC-Sign=F1B3F8371EA35936C053A6E2CF2998929111E0727FE1F39131F19E0937E792D8)
5. 左侧区域点击**New Query**后，右侧会出现**SQL Editor**页签，根据需要选择数据库，并输入SQL后，点击**Execute**按钮即可查看或修改数据。

修改数据后，点击SQL输入框下方的**Refresh Table**刷新页面上的数据。

 ![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/1d/v3/2S5XBw2vT2iay7fg__ws8g/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260420T194020Z&HW-CC-Expire=86400&HW-CC-Sign=6FD607FBD5D22EEF87793ACCB9C5614A962BA62E398D240E40BD37912F4FCF2C) 

  - 通过SQL修改数据或应用更新数据后，数据展示页面不支持自动刷新，需要重新执行查询语句或者点击刷新按钮。
  - 数据展示页面不支持可视化修改。
