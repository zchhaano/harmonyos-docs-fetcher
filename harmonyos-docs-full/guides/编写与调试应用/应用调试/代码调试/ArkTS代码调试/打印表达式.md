# 打印表达式

开发者可以通过 Evaluate and log 能力在代码执行到断点行时打印开发者指定的表达式。

1. 在需要打印表达式结果的地方设置断点。

![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226102043.46629771566206449210253654566661:50001231000000:2800:D4E802D817CE81533AC28DA8EC3831112E0776D7FE234755B61E754787F3D9E5.png)
2. 右键断点，然后点击**More**按钮。

![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226102044.27268797120541550939536507748584:50001231000000:2800:67B1DA19C13B9E3AC95BD1CFB4D02E6AC935963044BE921739984BAF850E91DC.png)
3. 勾选**Evaluate and log**复选框，并在下方输入框输入要打印的表达式。

![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226102044.24340415933384702676636412266929:50001231000000:2800:AB3A6ED1DC5D43CB0812EA628F4DD319BB308860D873BE44C42C43302C4A0860.png)
4. 启动调试，使断点命中，切换到调试的 Console 窗口，表达式的打印结果将在这里展示。
