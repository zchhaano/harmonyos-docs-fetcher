# 检查变量

当应用停止在某个断点处时，您可以在"Debugger"窗口中查看当前的变量信息。在"Frame"窗格中选择某个帧之后，可以在"Variable"窗格中检查变量，或对变量进行计算。

如需向"Watches"列表中添加变量或表达式，请按以下步骤操作：在"Watches"窗口中输入表达式，然后点击Add to Watches图标![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226102041.74574784926713194190827159599505:50001231000000:2800:790960A849A19C7CAA3476289CD7FBEBA74A68876C2D1C562E6BF5E246E30B21.png)。

如需从"Watches"列表中移除某一项，点击鼠标右键，选择**Remove Watches**。

![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226102041.81982525082042972322922811086327:50001231000000:2800:ABFCE78B026B7CA49D960475F946D7FBBAFA3F4330E0BAA29A20784485245A78.png)

从DevEco Studio 6.0.2 Beta1版本开始，支持预览pixelMap类型的变量。点击pixelMap变量右侧的**View PixelMap**，即可预览pixelMap。
