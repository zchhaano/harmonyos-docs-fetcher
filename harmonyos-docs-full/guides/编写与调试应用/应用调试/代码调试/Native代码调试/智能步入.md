# 智能步入

进行C++调试时，当前代码行有多个函数调用时，开发者可以使用Smart Step Into功能![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226102044.63228834913000976392461747152675:50001231000000:2800:CB7CCA5E397965A8F49852E1B48249BEC704936C6AF67FAB55D09088E3EEA3D2.png)直接Step Into到其中某一个函数的实现中。

## 操作步骤

通过点击调试窗口“entry-Native”调试器下的Debugger窗格中的按钮![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226102044.35574954508818231855233356057130:50001231000000:2800:417A66A24451F4EEC7D6566154734FED7A4EA3090AD3C909EFAD841E194D6A3A.png)（或使用快捷键**Shift+F7**）触发Smart Step Into功能后，DevEco Studio会将当前代码中可以进行跳转的函数进行高亮显示。

![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226102044.19886820322793328493764217033244:50001231000000:2800:4ECDFF3792E4A78D8310790004AB271151817F71966548F04179E51053913351.png)

开发者点击需要跳转的函数，程序会运行到目标函数的实现内。

 说明

已经执行完毕的函数不会高亮显示。
