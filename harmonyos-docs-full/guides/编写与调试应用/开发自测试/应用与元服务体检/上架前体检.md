# 上架前体检

 

从DevEco Studio 6.0.0 Beta1版本开始，AppAnalyzer新增上架前体检，针对上架阻塞问题进行快速检测，提前发现可能影响上架的问题，检测完成之后可以选择上传检测结果，用于应用市场上架参考，提升上架效率。

 

#### 前置操作

1. 通过以下任意一种方式，打开AppAnalyzer。

  - 单击菜单栏**Tools >****AppAnalyzer**，打开AppAnalyzer页面。
  - 在编辑窗口右侧的工具栏，点击**AppAnalyzer**或![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/c7/v3/V2qf8G_hT4qBAX-uOLkXCg/zh-cn_image_0000002561753015.png?HW-CC-KV=V1&HW-CC-Date=20260420T194021Z&HW-CC-Expire=86400&HW-CC-Sign=86CFB9483E96B26B953543E384ADC6B4773FF628A59425C4DE14404DC0DFF905)，打开AppAnalyzer页面。
2. 确保[DevEco Studio与真机设备已连接](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-run-device)，并对应用进行[签名](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-signing)。
3. 如果使用DevEco Studio 6.0.1版本，未配置Python环境时，请根据界面提示，下载Python及三方库。或者点击AppAnalyzer底部**Python 配置**按钮进行配置。

  

#### 进行体检

 

#### [h2]DevEco Studio 6.0.1 Beta1及以上版本

1. 在**AppAnalyzer**页面，选择**上架前体检**，点击预置的体检卡片，在弹框中选择待上架的产物和调试签名。

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/fe/v3/OtXcdhQfSmOszTQIXK0GFA/zh-cn_image_0000002530753082.png?HW-CC-KV=V1&HW-CC-Date=20260420T194021Z&HW-CC-Expire=86400&HW-CC-Sign=D4D7F6E8C957680047330CC68DC1FD9EAE91BCCA4844958707C2B6B41A4424CF)
2. 该体检模式无法自定义测试方式和体检规则，默认勾选所有规则，这些规则是[规则体检](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-app-analyzer-all-rules)的子集。单击底部的**开始体检**按钮，等待AppAnalyzer完成构建、签名、安装等操作。
3. 安装完成后，根据提示登录账号，开始进行测试。在测试过程中，请保持连接的设备为解锁亮屏状态。
4. 测试完成后，查看测试报告如下。

  - 如果测试分数是100分，可点击右上角**报告同步AG**按钮，弹出弹框，确认后点击**OK**，上传本次的检测报告，用于应用市场上架参考。上传报告后，无法再次上传报告。![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/e5/v3/-pcJpJ_nRq2fJaQjcPRMpw/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260420T194021Z&HW-CC-Expire=86400&HW-CC-Sign=ACC4758FA3BC0DDD0EC298FEFE69FEEFB3AB59A374810A73423CF9DF685865CF) 

如需上传报告，请在体检结束后上传，[历史报告](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-app-analyzer-history-reports)中无法上传报告。

  

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/6b/v3/Qsu07qUMT0Cpw-4fcL9TZA/zh-cn_image_0000002530753078.png?HW-CC-KV=V1&HW-CC-Date=20260420T194021Z&HW-CC-Expire=86400&HW-CC-Sign=749CDC0E3F57EBC182DC692DEA240BF8194B4E975506DCF9DF5EAC8901270EE2)
  - 如果测试分数不是100分，无法上传报告，可根据详情报告中的信息，对问题进行分析优化，详情报告的具体内容可参考[规则体检](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-app-analyzer-rules#li22241112508)。

 

从DevEco Studio 6.0.2 Beta1版本开始，如果在体检中遇到问题，可点击报告右上角的**用户反馈**向我们反馈。

 

从DevEco Studio 6.1.0 Release版本开始，支持导出报告，以实现报告的共享，具体可查看[导出报告](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-app-analyzer-history-reports#section78017171818)。

  

#### [h2]DevEco Studio 6.0.1 Beta1以下版本

1. 在**AppAnalyzer**页面，选择**上架前体检**，弹出上架前检测配置的弹框，选择待上架的产物和调试签名。

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/78/v3/Q5S0KL3HSGW7qifG26qHxQ/zh-cn_image_0000002561753017.png?HW-CC-KV=V1&HW-CC-Date=20260420T194021Z&HW-CC-Expire=86400&HW-CC-Sign=CF88AE91EC53C36B5AF3D6947A22BBEC5F07FB70267B6DCED7443252CCAF520C)

3. 安装完成后，根据提示登录账号，开始进行测试。在测试过程中，请保持连接的设备为解锁亮屏状态。
4. 测试完成后点击**结束**按钮停止测试任务，等待数据解析完成后，查看测试结果如下。

  - 如果测试分数是100分，可点击右上角**报告同步**按钮，弹出弹框，确认后点击**OK**，上传本次的检测报告，用于应用市场上架参考。上传报告后，无法再次上传报告。![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/7/v3/9MsmqbA6Q7quHfqscWRUEw/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260420T194021Z&HW-CC-Expire=86400&HW-CC-Sign=33428918E71B16F4C0FE23CA08C1461435683B77CC6E4028FBB2302A2E5FB734)
