# 设计稿代码生成

 

从DevEco Studio 6.1.0 Beta1开始，CodeGenie支持上传Pixso设计网站导出的设计稿，基于设计稿生成应用/元服务可用的代码，生成结果支持实时预览，帮助开发者快速完成页面搭建。

 

#### 操作步骤

1. 点击页面右侧菜单栏CodeGenie图标完成登录后，可以通过如下三种方式进入上传设计稿入口：

  - 在对话框输入"/"调出命令，选择**Generate UI Code****>****@Add Context****> UX Design Draft**。
  - 在对话框左下角下拉框选择**Generate UI Code >@Add Context > UX Design Draft**。
  - 在模块右键选择**New > Page > Generate Pages with AI >@Add Context > UX Design Draft**。

 

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/19/v3/6pjqp1VIQLCOis3hsOwjhw/zh-cn_image_0000002561752645.png?HW-CC-KV=V1&HW-CC-Date=20260420T193623Z&HW-CC-Expire=86400&HW-CC-Sign=DFE22C3F1D36ED42F882B2FF05A518F91AE8ACE715AD8CD78223AE36CF213635)
2. 上传.pxhm后缀的设计稿文件，勾选解析出的设计稿，点击**Submit**提交，最多勾选和提交5张设计稿。

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/6a/v3/f8XlocNWTQKrvMWn2mSDxA/zh-cn_image_0000002530752706.png?HW-CC-KV=V1&HW-CC-Date=20260420T193623Z&HW-CC-Expire=86400&HW-CC-Sign=33E08B9AEA883FC6279214BA991C457A55196A27A48680483CC47F2CCA8BA590)
3. 对生成的页面进行预览。若提交的多张设计稿的底部导航栏相同，预览时支持点击导航栏进行切换；若提交的多张设计稿底部导航栏不同，默认预览第一张，不支持点击切换。

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/bc/v3/fXlnnTy_R32FzgGTNnlwOQ/zh-cn_image_0000002561832625.png?HW-CC-KV=V1&HW-CC-Date=20260420T193623Z&HW-CC-Expire=86400&HW-CC-Sign=0FE921ED4CECC3DD5609078CFC7F04C9D0091FF755F89F389693620B3AD131D9)
4. 点击**Save to Project**，在**Module**选择保存的模块和在**Page name**设置目录名称，点击**Next**。

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/80/v3/3nYzuCSpQY-nWOLrGAqdrw/zh-cn_image_0000002530752710.png?HW-CC-KV=V1&HW-CC-Date=20260420T193623Z&HW-CC-Expire=86400&HW-CC-Sign=4322738F158EC4F570747087FA345B76FD1AED66D4221E8A3B0050D868FAB8F4)
5. 将生成的代码文件及资源保存至工程中，绿色文件为新增，蓝色文件为当前文档有修改，点击**Finish**完成添加。
