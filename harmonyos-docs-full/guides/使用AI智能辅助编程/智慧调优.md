# 智慧调优

DevEco Studio的Profiler工具中已集成智慧调优能力，支持通过自然语言交互，分析并解释当前实例或项目中存在的性能问题，帮助开发者快速定位影响性能的具体原因。

从DevEco Studio 6.0.0 Beta1版本开始，CodeGenie支持对Launch冷启动问题和Frame卡顿丢帧问题进行智慧调优分析。

从DevEco Studio 6.0.0 Beta3版本开始，CodeGenie支持对Allocation内存分配问题和Snapshot内存堆快照问题进行智慧调优分析。

从DevEco Studio 6.0.2 Beta1版本开始，CodeGenie增加了OOM内存溢出场景的分析能力，主要包括ArkUI组件、NAPI、闭包等内存问题场景。

## 操作步骤

1. 首次使用请先根据界面提示完成CodeGenie授权登录。当前支持如下两种开启方式：

**方式一：**若Launch、Frame、Allocation、Snapshot模板已录制完成，点击**Session**窗口中该条会话上的![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226101939.29648643045557113312411874346499:50001231000000:2800:CA5C67EED91A77C367D2159202A955153234AA37C091BAB1C49852ECC23D1DFB.png)图标，即可开始智慧调优分析。录制方法具体请参考[性能问题定位：深度录制](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/deep-recording)。

![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226101939.29144512615628386737465701450490:50001231000000:2800:CE79BE63FBCD4427FE7E294C576FAF61FAE72B069FFFC996BA894B548FEAC554.png)

**方式二：**切换至**Assistant**窗口，点击**Create Session**开始录制调优任务；或点击**Open File**按钮导入已有的调优数据文件，当前支持导入的文件类型包括.insight、.heapsnapshot、.rawheap。

![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226101939.71536453521807910860069491636511:50001231000000:2800:D9E69E497EBC1E4E79ECFE1AB4DA6E413A64872354E79903854E8ACCCEA77394.png)
2. 对于方式二，在**Assistant**页面，点击**Create Session**按钮，从**Launch**、**Frame**、**Snapshot**、**Allocation**中选择一个分析模板。

![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226101939.45442051253094652964535925796249:50001231000000:2800:87F18E3A7D8895CE77E33D5D5BDCC1C1CEA91CC805C19F80481B181F2D5F9FDC.png)

 说明

使用Snapshot模板对堆快照问题进行分析时，支持在对话框中选择单个Snapshot分析，或选择两个Snapshot进行对比分析。

![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226101939.22285643006160294189439670342195:50001231000000:2800:859219C001611DFD40B1E6CBD3E9C47200519CC28080EADD66AFB0083CC22DDD.png)
3. 以Allocation为例，录制新的调优任务或导入本地已有的调优数据模板文件。

![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226101939.48673707354898696147639819831029:50001231000000:2800:798540CFA142C336AFD88F7BDB90699B8EC7A3DDBEA48EEC7517B40E7EA3441F.png)
4. 等待AI完成初步分析。左键点击高亮的泳道名称，点击**Analyze**进一步分析该阶段的具体内存信息，点击**View Lane**在右侧查看具体的泳道信息。

![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226101939.88936115197298113696879801052824:50001231000000:2800:0985445C2A4D24F52FA3E9772C7D8D755A43588FE9B523C05B2601F2ED8BDEE0.png)
