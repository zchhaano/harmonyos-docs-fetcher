# 代码分析

CodeGenie支持对在**对话框中**输入对代码段和代码文件分析要求，帮助开发者快速理解代码逻辑、代码功能、技术细节和潜在问题等，提升开发效率。

DevEco Studio 6.0.2 Beta1之前版本，分析代码文件时需要通过**Files**入口选中文件；分析代码片段时，选中代码段后需点击![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226101944.70968787786607004610513078286537:50001231000000:2800:74B7C9571A46753BD853157DDD1ED0FE2EB29A143E12E3641F6C6270B3F69022.png)图标开启光标上下文功能。

在DevEco Studio 6.0.2 Beta1版本，分析代码文件时，支持在对话框输入要分析的代码文件或直接分析当前文件；分析代码片段时，选中代码段后直接分析，无需开启![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226101944.56674555071984491115065307291662:50001231000000:2800:52D6BF7DAE8349B3E33D03D665D93B0DC81841D21ECCB680BC1F42723A5780A1.png)图标。

从DevEco Studio 6.0.2 Release版本开始，使用HarmonyOS Ask智能体分析代码文件。

以DevEco Studio 6.0.2 Release和DevEco Studio 6.0.1 Beta1版本为例说明，操作如下。

**DevEco Studio 6.0.2 Release版本**

1. 选择HarmonyOS Act智能体，在对话框中输入**@**符号选择**Files**，或点击**@****Add Context** > **Files**，选择需要分析的代码文件，或在对话框输入文件路径指定需要分析的代码文件，或分析当前代码文件。
2. 在对话框输入描述，点击![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226101944.32023979768991171849681400871422:50001231000000:2800:98F22F38622ABB99644E77712A5756D8D4B64003EB611FBB7FB9A5C6DFB0BDC1.png)发送后等待回复。

**DevEco Studio 6.0.1 Beta1版本**

分析代码文件：在对话框中输入**@**符号选择**Files**，或点击**@****Add Context** > **Files**，可对单个或多个代码文件进行分析。

分析代码片段：选中代码段，点击![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226101944.13884950564894770008935240878111:50001231000000:2800:3766BB412E7D8DF00A760D485E6D9482B7407DFE2B59CD9309DA64E7A3F7E579.png)图标开启光标上下文功能，在对话框内描述，开始分析。
