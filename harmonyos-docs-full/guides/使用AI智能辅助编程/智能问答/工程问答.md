# 工程问答

 

从DevEco Studio 6.1.0 Beta2版本开始，CodeGenie新增工程问答的功能。工程问答通过分析用户代码库，快速定位文件、类、函数、变量、常量、UI元素等的具体位置，对开发者提出的本地工程代码检索类问题作出精准回答，并完成具体的代码解读。

 

 

选择**HarmonyOS Ask**智能体，在对话区域输入“**/**”后，在弹出的菜单中选择“**Project Ask**”，在对话框中输入本地工程代码检索类问题，点击![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/70/v3/cJv8aykUSz6SlRrubS-UXw/zh-cn_image_0000002530753342.png?HW-CC-KV=V1&HW-CC-Date=20260420T193622Z&HW-CC-Expire=86400&HW-CC-Sign=AA66B4BEFD3323BEB978B20AE98CC30C8E53112361F297F5DD350EE484E09ADF)发送后等待回复：
