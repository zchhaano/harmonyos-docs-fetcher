# 编辑区对话

 

CodeGenie提供Inline Edit能力，支持在ArkTS文件的编辑窗口中通过自然语言进行问答，基于上下文智能生成代码片段，提升代码可读性。

 

从DevEco Studio 6.0.2 Beta1开始，Inline Edit支持选择三方模型，根据指定的模型进行生成代码。

 

从DevEco Studio 6.1.0 Beta1开始，Inline Edit入口名称变更为Inline Chat。

 

从DevEco Studio 6.1.0 Beta2版本开始，需要在设置中勾选Show Inline Chat tips开启编辑区Inline Chat浮窗。

 

 

1. 当前有以下两种方式唤醒Inline Chat对话框：

  - 若未选中代码片段，在代码编辑区域右键选择**CodeGenie > Inline Chat**（或使用快捷键**Alt+I**，macOS中为**Command+I**）。

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/40/v3/m9mu1gDRSK2zNx5AqHUpww/zh-cn_image_0000002530753294.png?HW-CC-KV=V1&HW-CC-Date=20260420T193622Z&HW-CC-Expire=86400&HW-CC-Sign=498E891817418BC0EBE8D7FFE98F6D165EC1B66940CB0D8E36691112071513AC)
  - 若选中一段代码，点击**Inline Chat（**或使用快捷键**Alt+I**，macOS中为**Command+I）**浮框。

在DevEco Studio 6.1.0 Beta2之前版本，如未出现浮框，可在**File** > **Settings** > **CodeGenie**> **Code Generation**（macOS中为**DevEco Studio** > **Prefe****rences/Settings**> **CodeGenie**>**Code Generation**）中取消勾选**Hide  Inline Chat Overlay**选项。

 

从DevEco Studio 6.1.0 Beta2版本开始，如未出现浮框，可在**File** > **Settings** > **CodeGenie > Code Completion****& Inline Chat**（macOS中为**DevEco Studio** > **Prefe****rences/Settings**> **CodeGenie**>**Code Completion & Inline Chat**）中勾选**Show Inline Chat tips**启用浮窗。

 

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/f2/v3/4_NrRaekQrmvWPZhmyx_gQ/zh-cn_image_0000002530913284.png?HW-CC-KV=V1&HW-CC-Date=20260420T193622Z&HW-CC-Expire=86400&HW-CC-Sign=6A16A293C415BDC7E507F8938A53245B4CFB4C35DE88D6EABC9AFFAE8E5F73EC)
2. 选择在CodeGenie中已配置的三方模型，或者使用默认模型。三方模型配置具体请参考[模型（Model）配置](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-agent-model)。

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/0d/v3/UC9ZhoqNQxiI86N6x5ll7g/zh-cn_image_0000002530753296.png?HW-CC-KV=V1&HW-CC-Date=20260420T193622Z&HW-CC-Expire=86400&HW-CC-Sign=41D091CB199B5873AA940EC40F1AAE0080F57D9E7BF6B1CC5C22DA3ED7992415)
3. 若选择默认模型，在对话框中输入所需要的代码功能描述，在键盘输入回车或点击发送，开始生成代码。点击**Stop Generation**，中断本轮代码生成过程。

若选择三方模型，支持分析当前代码文件和生成分析报告，以及进行参数校验（**Parameter Validation**）、代码注释（**Code Explanation**）、代码优化（**Code Optimization**），分析报告和参数校验等结果跟模型有关，具体操作如下：

  - 未选中代码片段，在对话框中输入"/"，在键盘输入回车或点击发送，对当前代码文件开始分析。点击**Stop Generation**，中断本轮代码生成过程。

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/d2/v3/q1wIOQzkSemcAIYTyx8DZQ/zh-cn_image_0000002561753231.png?HW-CC-KV=V1&HW-CC-Date=20260420T193622Z&HW-CC-Expire=86400&HW-CC-Sign=E5ADB809440C9E936FFFF13338124517BEA8089C4AE1724EEFF015AD98F90B48)
  - 选中一段代码，在对话框中输入"/"，选择**Parameter Validation**/**Code Explanation**/**Code Optimization**，可输入或不输入所需的功能描述，在键盘输入回车或点击发送后开始生成。点击**Stop Generation**，中断本轮代码生成过程。

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/9d/v3/2RDUpbv2S6ayGdmQc7RXYA/zh-cn_image_0000002561833211.png?HW-CC-KV=V1&HW-CC-Date=20260420T193622Z&HW-CC-Expire=86400&HW-CC-Sign=10824CA305EE6A440D4000B8E634B6538F92504B13D19A1FC43CE66DA8E48FA6)
4. 生成完毕将在编辑区展示本轮生成的代码内容，并通过不同颜色体现与当前代码的对比差异。

  - 绿色区域：新生成的代码内容。
  - 蓝色区域：对现有代码进行修改的内容。
  - 红色区域：删除的代码内容。

 ![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/de/v3/sPw6HCL9SlKr-hsSpp9F_Q/zh-cn_image_0000002530913290.png?HW-CC-KV=V1&HW-CC-Date=20260420T193622Z&HW-CC-Expire=86400&HW-CC-Sign=CF5FACA81F6B02D8682B5B95D4E7350E10406E1E569B955F85B21C9D657146E7)

  - 点击Inline Chat对话框中**Accept All**（或使用快捷键**Alt+Enter**），接受当前生成的全部内容；
  - 点击Inline Chat对话框中![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/7c/v3/9qhBKtHKTbax1un0wtZXyQ/zh-cn_image_0000002561833215.png?HW-CC-KV=V1&HW-CC-Date=20260420T193622Z&HW-CC-Expire=86400&HW-CC-Sign=AC3AC28F5C725B0219630C5E4EA233AEBB922B4B49F73A4BEB9E07A0EC87CA7F)刷新按钮**/****Regenerate**，将根据当前描述重新生成代码片段；
  - 点击编辑区中**Accept**（或使用快捷键**Shift+Ctrl+Y**，macOS上为**Shift+Command+Y**），分段逐一接受并保留生成内容；
  - 点击编辑区中**Reject**（或使用快捷键**Shift+Ctrl+N**，macOS上为**Shift+Command+N**），分段逐一拒绝并删除当前生成内容；
  - 点击**Further Edit**（或使用快捷键**Ctrl+K**，macOS上为**Command+K**），重新进行输入，开始新一轮问答。
