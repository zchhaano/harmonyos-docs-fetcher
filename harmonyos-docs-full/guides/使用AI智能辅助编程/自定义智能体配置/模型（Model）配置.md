# 模型（Model）配置

CodeGenie支持通过Anthropic-API、Gemini-API和OpenAI-API协议接入第三方模型，为自定义Agent提供多样化的模型选择。

从DevEco Studio 6.0.1 Beta1开始，CodeGenie支持通过OpenAI-API协议接入第三方模型。

从DevEco Studio 6.0.2 Beta1开始，CodeGenie支持通过Anthropic-API、Gemini-API协议接入第三方模型，以及新增Built-in Models内置模型。

## 操作步骤

1. 点击界面右上方**Settings**![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226101945.03075480790554657935573893690127:50001231000000:2800:28A245DD4E11C51C4F5A1E7A34778A41A4A15304094F5E56F114BA23CE4A6AA6.png)按钮，选择**Model**，进入配置页面。

![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226101945.97319846968936390681512788165903:50001231000000:2800:67B8C23731B614DCD9B0D02AB3B3CF8948D3835885A63571E7A9D014598EA218.png)
2. 点击![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226101945.09945757599158177544667966754628:50001231000000:2800:5985E48E0DAB0C0E6CE15674A6B5F55B57DB900F3BE08C64AFA12FC12396F871.png)按钮，填写相关信息。点击**Add**，校验成功的模型将被添加到列表中。

  - **Name**：模型名称。
  - **Protocol**：模型的协议。
  - **Url**：模型的访问地址。
  - **API Key**：模型的访问密钥。
  - **Model**：模型的标识。

![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260226101945.98467694984438654955320262266594:50001231000000:2800:5726B927240E4A7B12A9CA051E3C182661A2AEA92D823A61CAE24F634574FE7B.png)
3. 在**All Models**下展示所有添加成功的模型，Built-in Models为内置模型，Custom Models为自定义模型。将鼠标悬浮在自定义模型上会显示两个操作按钮：编辑、删除，方便开发者管理自定义模型。
