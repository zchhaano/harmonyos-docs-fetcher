# 个人数据处理说明

此文档针对华为作为最终用户数据处理者，开发者作为最终用户数据控制者的数据处理进行说明，包括：

- 华为处理的个人数据清单
- 指导开发者如何帮助最终用户实现对数据的控制

## 华为处理的个人数据清单

最后修改时间：2025/05/20

 展开

| 个人数据清单 | 使用目的 | 存留期 |
| --- | --- | --- |
| 个人基本资料 | 通过智能填充服务获取姓名用于给用户提供表单填充推荐。 | 1、在表单中填写过的信息保存至历史表单后，可前往“设置 > 隐私和安全 > 智能填充 > 历史表单输入”删除保存的历史表单输入记录。 2、可前往“设置”>“隐私和安全”>“智能填充”关闭“华为账号”开关，在“智能填充”缓存的华为账号相关数据会自动删除。 |
| 账号信息 | 通过智能填充服务获取昵称用于给用户提供表单填充推荐。 |  |
| 联系方式 | 通过智能填充服务获取手机号码、邮箱号用于给用户提供表单填充推荐。 |  |
| 位置信息 | 通过智能填充服务获取粗略位置信息用于给用户提供表单填充推荐。 |  |
| 身份信息 | 通过智能填充服务获取身份证、护照信息用于给用户提供表单填充推荐。 |  |
| 应用信息 | 通过智能填充服务获取应用基本信息用于保存智能填充使用记录，并用于填充建议排序。 |  |
| 日程/日历信息 | 通过智能填充服务获取日程信息用于给用户提供表单填充推荐。 |  |
| 通讯信息 | 通过智能填充服务获取通讯录信息用于给用户提供表单填充推荐。 |  |
| 车牌信息 | 通过智能填充服务获取车牌信息用于给用户提供表单填充推荐。 |  |
| 设备信息 | 当用户在三方应用使用智能填充功能时，将获取设备信息以提供产品运营与维护服务。 |  |
| 消费信息 | 通过智能填充服务获取发票信息用于给用户提供表单填充推荐。 |  |

**指导开发者如何帮助最终用户实现对数据的控制**

1. 开发者通过智能填充服务获取的用户数据，需要开发者自行提供对应的数据主体权利。
2. 智能填充服务存储的用户个人数据请参考《[关于智能填充与隐私的声明](https://legal.cloud.huawei.com/terms/scope/huawei/autofill/privacy-statement.htm?code=CN&language=zh-CN&version=20250521)》。
