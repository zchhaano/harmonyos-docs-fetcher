# 剪贴板粘贴框遮挡智能填充选择框

**现象描述**

![image](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20260224165608.70485471013828523972447221123742:50001231000000:2800:8AE06A569F2FFE5565B006D6A85924523E609EC80AA1ABAD3D97855FBE8B19AA.jpg)

**解决措施**

在代码文件中设置.selectionMenuHidden(true)，使剪贴板粘贴框隐藏。

 收起自动换行深色代码主题复制

```
Row () { Text ( '收货人：' ). textAlign ( TextAlign . End ). width ( '25%' ) TextInput (). width ( '75%' ). contentType ( ContentType . PERSON_FULL_NAME ). selectionMenuHidden ( true ) }
```
