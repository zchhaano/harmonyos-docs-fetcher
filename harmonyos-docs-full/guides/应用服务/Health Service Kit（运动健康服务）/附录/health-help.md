# 如何获取帮助

 

建议优先查找官方文档，快速获取所需指导。查阅文档后，仍不能解决的问题，可以通过以下方式进行反馈。

 

- 通过[在线提单](https://developer.huawei.com/consumer/cn/support/feedback/#/)反馈问题。提交问题后，通常会在1~2个工作日内收到回复。有时需要您进一步澄清问题，请及时关注进展并予以回复，以便更好地解决问题。
- 通过[华为开发者论坛](https://developer.huawei.com/consumer/cn/forum/block/huawei-hihealth)反馈问题。通过华为开发者论坛中的[运动健康](https://developer.huawei.com/consumer/cn/forum/block/huawei-hihealth)板块反馈的相关问题，通常会在1~2个工作日内收到回复。
- 通过官网文档反馈问题。

 

  1. 选中您存在疑问的文档内容，单击意见反馈按钮。
  2. 选择问题分类，并对问题进行描述，并单击“确定”提交。![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/ba/v3/FYPJwt2LRVqikBb-hOeDUg/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260420T191235Z&HW-CC-Expire=86400&HW-CC-Sign=BB7EE09F373359B55825F0F6E8293AB4F152652B47D3E82EC9E0BE098456F3F1)  

    - 文档修改和版本更新发布需要一定的周期，非紧急问题可以通过该途径反馈。
    - 文档反馈系统不支持问题答复。问题描述得越清晰，将越有可能得以解决。
