# 申请优先通知权益

  

当用户终端收到携带[priorityNotificationType](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-inner-notification-notificationrequest)字段的通知消息时，系统会将其识别为优先通知并优先显示。

 

申请优先通知权益存在以下限制：

 

1）该优先通知消息仅对商务办公、社交类应用开放。

 

2）应用仅在重要联系人、@我、加急消息、重要日程类提醒等场景下使用该能力。

   

#### 申请步骤

 

1. 登录[AppGallery Connect](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html)，选择“开发与服务”。

 

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/8a/v3/6r8nO_6ZRwSQQBf4CLep9g/zh-cn_image_0000002573974993.png?HW-CC-KV=V1&HW-CC-Date=20260420T191247Z&HW-CC-Expire=86400&HW-CC-Sign=31EDCF1A49CFBE07FC8ACB21EA54A0F5C55795EC7056ED7A0DE5C5C3A7C92041)
2. 在项目列表中找到您的项目，在项目下的应用列表中选择需要申请优先通知权益的应用。

 

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/8d/v3/mPnlgZbWTJCKK-k_ritI-Q/zh-cn_image_0000002543374760.png?HW-CC-KV=V1&HW-CC-Date=20260420T191247Z&HW-CC-Expire=86400&HW-CC-Sign=3BAC7C735447B13D10DE50FEA8E2372AE8A85575D2E0DFB77A3D9FDA02A0E06C)
3. 进入“项目设置 > 开放能力管理”页面，点击“优先通知”的“申请”。

 

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/34/v3/FHgV0zixRrO39k8O5O9GBw/zh-cn_image_0000002543215098.png?HW-CC-KV=V1&HW-CC-Date=20260420T191247Z&HW-CC-Expire=86400&HW-CC-Sign=4482B3A43C9EE2843C13C1A9F31DC09792119A157B30ED71BB0255B689BADFA4)
4. 开发者可参考“申请原因”中的模板，提供申请必须的相关信息，包括应用介绍、使用场景、申请用途、附件、承诺信息，然后点击“提交”按钮。

 

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/aa/v3/x1y67k4FQc29XMU-v3WWfQ/zh-cn_image_0000002573855013.png?HW-CC-KV=V1&HW-CC-Date=20260420T191247Z&HW-CC-Expire=86400&HW-CC-Sign=98D9D076910BBE5D71A43F0EC257BE7B030AF4FFFB722C9BE016C9FE2C108613)
5. 开发者可通过互动中心的“服务开通申请”消息获取优先通知权益申请结果。

 

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/6c/v3/-sspAypRSYCj5rNJGzHyRw/zh-cn_image_0000002573974995.png?HW-CC-KV=V1&HW-CC-Date=20260420T191247Z&HW-CC-Expire=86400&HW-CC-Sign=26542F788011A498333B3524A64449E4981AAA0D16ABE503315F20A65DF2DA64)

 

![image](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/89/v3/zx_3uQDPQQ2h9Wwf-nvVVA/zh-cn_image_0000002543374762.png?HW-CC-KV=V1&HW-CC-Date=20260420T191247Z&HW-CC-Expire=86400&HW-CC-Sign=B124C1723D6488B1FA72F27BC7E2DE8C7F08AC714525336A54023542BD6787B3)
6. 优先通知权益申请通过后，须在“证书、APP ID和Profile”页面下左侧树形菜单的“Profile”页签，点击“添加”重新生成Profile文件，并下载Profile文件到本地，然后在“[发布应用](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-publish-app)”时，须将该Profile打包到应用包中。
