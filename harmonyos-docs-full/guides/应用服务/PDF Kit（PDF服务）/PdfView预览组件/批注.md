# 批注

进入批注模式，目前支持高亮、下划线和删除线类型批注。

## 接口说明

  展开

| 接口名 | 描述 |
| --- | --- |
| enableAnnotation (annotationType: SupportedAnnotationType, color?: number): void | 在常用操作之间切换并添加批注。 |

## 示例代码

1. 先加载PDF文档。
2. 调用PdfView预览组件，渲染显示。
3. 调用enableAnnotation方法，进入批注模式。

 收起自动换行深色代码主题复制
