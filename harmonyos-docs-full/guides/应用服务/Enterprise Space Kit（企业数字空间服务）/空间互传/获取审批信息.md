## 场景介绍

Enterprise Space Kit为应用提供获取审批信息的能力。文件外发需经过审批流程控制，通过调用审批状态同步接口实时获取审批结果，审批完成后允许文件外发至个人空间，若审批被拒绝或撤销则禁止外发。

## 接口说明

详细接口说明可参考[接口文档](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/enterprisespace-spacedatatransfer#section5740142017119)。

  展开

| 接口名 | 描述 |
| --- | --- |
| getAuditInfo (transactionNum: string): AuditInfo | 获取审批信息。 |

## 开发步骤

1. 导入Enterprise Space Kit模块。 

 收起自动换行深色代码主题复制

```
import { fileTransfer } from '@kit.EnterpriseSpaceKit' ;
```
2. 调用[getAuditInfo](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/enterprisespace-spacedatatransfer#section5740142017119)接口，获取审批信息，并且查看打印信息。
