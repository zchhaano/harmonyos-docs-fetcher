# 个人数据处理说明

此文档针对华为作为最终用户数据处理者，开发者作为最终用户数据控制者的数据处理进行说明，包括：

- Game Service Kit处理的个人数据清单
- 指导开发者如何帮助最终用户实现对数据的控制

## Game Service Kit处理的个人数据清单

最后修改时间：2025/08/26

 展开

| 个人数据清单 | 使用目的 | 存留期 |
| --- | --- | --- |
| 社交信息 | 保障用户正常启动/登录游戏。 | 数据产生后在未销户的情况下最多存储2年（由当地法律法规决定），超期将自动删除。 |
| 位置信息 | 满足相关法律法规对实名制的要求。 | 使用后立即删除，不涉及存留期。 |
| 身份信息 | 满足相关法律法规对实名制的要求。 | 用户如果销户，则30天内自动删除。 |
| 设备信息 | 满足相关法律法规对实名制的要求，对未成年用户提供防沉迷保护。 | 使用后立即删除，不涉及存留期。 |
| 应用信息 | 保障用户正常启动/登录游戏，对未成年用户提供防沉迷保护。 | 数据产生后在未销户的情况下最多存储7年（由当地法律法规决定），超期将自动删除。 用户如果销户，则30天内自动删除。 |
| 账号信息 | 保障用户正常启动/登录游戏，满足相关法律法规对实名制的要求，对未成年用户提供防沉迷保护。 | 用户如果销户，则30天内自动删除。 |
| 应用信息（应用设置信息、应用基本信息） | 用于游戏场景感知上报游戏信息。 | 数据产生后在未销户的情况下最多存储180天（由当地法律法规决定），超期将自动删除。 |
| 用户应用基本信息（应用包名、客户端版本号） | 用于游戏近场快传上报游戏信息。 | 数据不上云，不涉及存留期。 |

## 指导开发者如何帮助最终用户实现对数据的控制

**如何清除最终用户的数据**

最终用户删除华为账号且操作成功后，Game Service Kit会清除最终用户登录记录中能够绑定到最终用户的数据。

Game Service Kit最多保存最近5年（由当地法律法规决定）的登录记录，超期登录记录会自动删除。
