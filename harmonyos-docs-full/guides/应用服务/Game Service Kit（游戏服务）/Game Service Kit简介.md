## 业务简介

Game Service Kit（游戏服务）主要提供快速、低成本构建游戏基本能力与游戏场景优化服务，有效提升游戏开发效率，帮助游戏运营。

 展开

| 功能 | 描述 |
| --- | --- |
| 基础游戏服务 （必选） | 为游戏快速、低成本地构建基础功能，例如联合登录、华为账号实名认证、未成年人防沉迷等，让开发者聚焦游戏本身的业务能力，从而迅速推广游戏，并基于用户和内容的本地化进行深度的游戏运营。 |
| 游戏场景感知 （可选） | 通过游戏为系统提供精细化场景信息、配置信息、网络信息等数据，系统向游戏反馈系统状态等信息，使得双方能够利用这些信息进行更紧密和深入的协作，在系统资源有限的情况下优化玩家的游戏体验。 |
| 游戏近场快传 （可选） | 游戏近场快传服务支持设备在彼此靠近的情况下进行游戏数据交换。 |

## 约束与限制

### 支持的国家和地区

本Kit仅支持华为账号的服务地和注册地均为中国境内（香港特别行政区、澳门特别行政区、中国台湾除外）的用户。

### 支持的设备

 展开

| 功能 | 支持的设备 |
| --- | --- |
| 基础游戏服务 | Phone、Tablet、PC/2in1、TV |
| 游戏场景感知 | Phone、Tablet、PC/2in1 |
| 游戏近场快传 | Phone、Tablet、PC/2in1 |

## 模拟器支持情况

本Kit暂不支持模拟器。
