# 个人数据处理说明

此文档针对华为作为最终用户数据处理者，开发者作为最终用户数据控制者的数据处理进行说明，包括：

- 华为处理的个人数据清单
- 指导开发者如何帮助最终用户实现对数据的控制

## 华为处理的个人数据清单

最后修改时间：2024/05/20

  展开

| 个人数据清单 | 使用目的 | 存留期 |
| --- | --- | --- |
| 设备标识符 | 设备标识符，基于该标识符记录设备的应用使用状态信息。 | DeviceVerify服务器的数据存储时间不超过2年，超期信息自动删除。 |
| 应用使用信息 | 记录用户在该设备使用应用的状态（如是否领取优惠券）。 |  |
| 设备标识符 | 设备标识符，基于该标识符记录设备中存在风险的使用信息 。 | 业务风险检测服务器的数据存储时间不超过180天，超期信息自动删除。 |
| 设备使用信息 | 记录用户在使用设备过程中的风险信息（例如：是否接听诈骗电话、是否正在共享屏幕等），用于业务风险检测。 |  |
| ODID | ODID，基于ODID记录设备中存在风险的使用信息。 |  |
| 通话记录信息 | 包括来电方号码、接听通话的时间戳、通话记录类型、来电方名称以及通话时长。上述信息将传递给第三方应用，用于实现诈骗通话记录举报功能。 | 不留存。 |
| 消息信息 | 包括消息发送方号码、接收消息的时间戳、消息内容、发送方名称、消息发送方号码归属地、消息类型、彩信主题、彩信附件类型、附件URI。上述信息将传递给第三方应用，用于实现诈骗消息举报功能。 | 不留存。 |

## 指导开发者如何帮助最终用户实现对数据的控制

- 如何清除最终用户的数据       

  - 开发者调用DeviceVerify服务器的[删除设备标记状态接口](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/devicesecurity-deviceverify-deletedevicestatus)主动删除用户的数据。
- 如何导出最终用户的数据       

  - 开发者调用DeviceVerify服务器的[查询设备标记状态接口](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/devicesecurity-deviceverify-getdevicestatus)导出用户的数据。
