## 应用或元服务动态内存峰值占用

| 描述 | 建议应用完成操作后，各类应用在后台的内存占用峰值应≤ 1000MB。 |
| --- | --- |
| 类型 | 建议 |
| 适用设备 | 手机，平板，智慧屏，车机，手表 |
| 应用形态适用性 | 鸿蒙应用，鸿蒙元服务 |
| 说明 | 无 |

| 应用形态适用性 | 鸿蒙应用，鸿蒙元服务 |
| 说明 | 无 |
