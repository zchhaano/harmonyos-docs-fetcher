# OH_AVSamplesBuffer

```
typedef struct OH_AVSamplesBuffer OH_AVSamplesBuffer
```

## 概述

支持设备Phone

LowPowerAVSink输入数据的结构体。应用在收到DataNeeded回调后需要将数据打包装进OH_AVSamplesBuffer实例中送给对应的lowpower_avsink。

**起始版本：** 20

**相关模块：** [AVSinkBase](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-avsinkbase)

**所在头文件：** [lowpower_avsink_base.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-lowpower-avsink-base-h)
