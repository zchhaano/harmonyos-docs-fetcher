# OH_AVMediaSourceLoadingRequest

 

```
typedef struct OH_AVMediaSourceLoadingRequest OH_AVMediaSourceLoadingRequest

```

 

#### 概述

加载请求对象，用于表示媒体资源的加载请求，应用通过该对象获取所请求资源的位置。

 

**起始版本：** 23

 

**相关模块：** [avmedia_source](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-avmedia-source)
