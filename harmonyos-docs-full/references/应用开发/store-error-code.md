## 1009300115 当前adTechId下设置了过多的回传数据

支持设备PhoneTablet

**错误信息**

**错误描述**

当前adTechId下设置了过多的回传数据。

**可能原因**

1. 调用设置归因回传接口时触发过载防护校验，单个adTechId下，待回传的调试postback数量超过5个。

**处理步骤**

1. 按照单个adTechId下，待回传的调试postback数量不能超过5个规则调用接口。

2. 设置待回传postback数量不超过100条。
