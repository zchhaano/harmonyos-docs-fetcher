# RemoteLocationExtensionContext（定位扩展Context）

RemoteLocationExtensionContext是[RemoteLocationExtensionAbility](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/remote-location-ability)的上下文环境，继承自[ExtensionContext](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-inner-application-extensioncontext)。

**模型约束：**属性仅可在Stage模型下使用。

**系统能力：**SystemCapability.Push.PushService

**起始版本：**4.1.0(11)

## 导入模块

支持设备PhonePC/2in1TabletTVWearable收起自动换行深色代码主题复制

```
import { RemoteLocationExtensionContext } from '@kit.PushKit' ;
```

## RemoteLocationExtensionContext

支持设备PhonePC/2in1TabletTVWearable

**模型约束：**属性仅可在Stage模型下使用。

**系统能力：**SystemCapability.Push.PushService

**设备行为差异**：该属性在Phone、PC/2in1、Tablet中可正常使用，在其他设备类型中无效果。

**起始版本：**4.1.0(11)

本类继承自[ExtensionContext](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-inner-application-extensioncontext)，未新增内容。
