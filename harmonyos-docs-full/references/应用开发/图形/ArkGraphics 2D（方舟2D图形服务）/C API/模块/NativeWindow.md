## 概述

支持设备PhonePC/2in1TabletTVWearable

提供NativeWindow功能，作为数据生产者，可用来和egl对接。

**起始版本：** 8

## 文件汇总

 支持设备PhonePC/2in1TabletTVWearable展开

| 名称 | 描述 |
| --- | --- |
| buffer_handle.h | 定义NativeWindow模块使用的BufferHandle的结构体。引用文件：<native_window/buffer_handle.h> |
| external_window.h | 定义获取和使用NativeWindow的相关函数。引用文件：<native_window/external_window.h> |
| graphic_error_code.h | 定义错误码。引用文件：<native_window/graphic_error_code.h> |
