## 概述

支持设备PhoneTablet

此结构体描述2D图像分辨率，以像素为单位。

**起始版本：** 5.0.0(12)

**相关模块：** [GraphicsAccelerate](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/_graphics_accelerate)

## 汇总

支持设备PhoneTablet 

### 成员变量

 支持设备PhoneTablet展开

| 名称 | 描述 |
| --- | --- |
| uint32_t width | 图像宽度，以像素为单位。 |
| uint32_t height | 图像高度，以像素为单位。 |
