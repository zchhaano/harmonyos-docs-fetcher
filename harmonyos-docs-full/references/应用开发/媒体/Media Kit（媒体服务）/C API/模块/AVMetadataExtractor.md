## 概述

 支持设备PhonePC/2in1TabletTVWearable

提供从媒体资源中获取元数据的API。

根据实际的开发需求，参考对应的开发指南及样例[使用AVMetadataExtractor获取元数据](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/using-ndk-avmetadataextractor-for-media)。

**系统能力：** SystemCapability.Multimedia.Media.AVMetadataExtractor

**起始版本：** 18

## 文件汇总

 支持设备PhonePC/2in1TabletTVWearable 展开

| 名称 | 描述 |
| --- | --- |
| avmetadata_extractor.h | 定义AVMetadataExtractor接口。使用其C API从媒体资源中获取元数据。 |
| avmetadata_extractor_base.h | 定义AVMetadataExtractor常量。 |
