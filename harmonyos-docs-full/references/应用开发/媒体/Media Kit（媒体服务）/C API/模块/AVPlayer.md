## 概述

 支持设备PhonePC/2in1TabletTVWearable

提供用于播放媒体源的API。

根据实际的开发需求，参考对应的开发指南及样例。

- [使用AVPlayer播放音频](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/using-ndk-avplayer-for-playback)
- [使用AVPlayer播放视频](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/using-ndk-avplayer-for-video-playback)

**起始版本：** 11
