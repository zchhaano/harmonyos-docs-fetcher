# @ohos.app.form.FormEditExtensionAbility  (FormEditExtensionAbility)

FormEditExtensionAbility模块提供卡片编辑功能，继承自[UIExtensionAbility](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-app-ability-uiextensionability)。

 说明 

本模块首批接口从API version 18开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。

本模块接口仅可在Stage模型下使用。

## 导入模块

支持设备PhonePC/2in1TabletTVWearable收起自动换行深色代码主题复制

```
import { FormEditExtensionAbility } from '@kit.FormKit' ;
```

## FormEditExtensionAbility

支持设备PhonePC/2in1TabletTVWearable

提供卡片编辑功能。
