# OH_MediaAssetManager

收起自动换行深色代码主题复制

```
typedef struct OH_MediaAssetManager OH_MediaAssetManager
```

## 概述

支持设备PhonePC/2in1TabletTV

定义媒体资产管理器。

此结构提供了请求媒体库资源的能力。

如果创建失败，则返回空指针。

**起始版本：** 12
