# Input_CustomCursor

```
typedef struct Input_CustomCursor Input_CustomCursor
```

## 概述

支持设备PhonePC/2in1TabletTVWearable

自定义鼠标光标像素图资源。

**起始版本：** 22

**相关模块：** [input](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-input)

**所在头文件：** [oh_input_manager.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-oh-input-manager-h)

**相关接口：**

 展开

| 名称 | 描述 |
| --- | --- |
| OH_Input_CustomCursor_Create | 创建自定义鼠标光标资源对象。通过调用 OH_Input_CustomCursor_Destroy 销毁自定义鼠标光标资源对象。 |
| OH_Input_CustomCursor_Destroy | 销毁自定义鼠标光标资源对象。 |
