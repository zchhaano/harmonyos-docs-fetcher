# CallerInfoQueryExtensionContext (来去电信息查询扩展Context)

CallerInfoQueryExtensionContext为CallerInfoQueryExtensionAbility的上下文环境，继承自[ExtensionContext](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-inner-application-extensioncontext)。

**起始版本：**5.0.2(14)

## 导入模块

支持设备PhonePC/2in1TabletWearable

```
import { CallerInfoQueryExtensionContext } from '@kit.CallServiceKit' ;
```

## CallerInfoQueryExtensionContext
