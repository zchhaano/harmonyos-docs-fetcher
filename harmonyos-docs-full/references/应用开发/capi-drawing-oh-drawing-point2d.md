# OH_Drawing_Point2D

```
typedef struct {...} OH_Drawing_Point2D
typedef OH_Drawing_Point2D OH_Drawing_Corner_Radii
```

## 概述

支持设备PhonePC/2in1TabletTVWearable

OH_Drawing_Point2D: 定义一个二维的坐标点。

OH_Drawing_Corner_Radii: 定义一个圆角半径，该圆角半径由x轴方向和y轴方向上的半径组成。

**起始版本：** 12

**相关模块：** [Drawing](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-drawing)

**所在头文件：** [drawing_types.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-drawing-types-h)

## 汇总

支持设备PhonePC/2in1TabletTVWearable 

### 成员变量

 支持设备PhonePC/2in1TabletTVWearable展开

| 名称 | 描述 |
| --- | --- |
| float x | x轴坐标 |
| float y | y轴坐标 |
