# 逻辑组件错误码

以下仅介绍本模块特有错误码，通用错误码请参考[通用错误码说明文档](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/errorcode-universal)。

     

#### 103801 ForEach id生成失败

 

**错误信息**

 

use of default id generator function not possible on provided data structure.Need to specify id generator function (ForEach 3rd parameter).Application Error!

 

**错误描述**

 

[Foreach](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-rendering-control-foreach)使用默认的键值生成函数无法基于提供的数据结构生成键值。

 

**可能原因**

 

开发者提供的数据源无法生成键值，例如数据项类型有问题。

 

**处理步骤**

 

修改数据源对象或者开发者手动实现一个键值生成函数，请参考[键值生成规则](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-rendering-control-foreach#键值生成规则)

    

#### 103802 渲染子节点异常

 

**错误信息**

 

lacks mandatory '.each' attribute function, i.e. has no default item builder. Application error!

 

**错误描述**

 

缺少'[each](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-rendering-control-repeat#each)'属性。

 

**可能原因**

 

缺少'each'属性。

 

**处理步骤**

 

设置'each'，提供默认的组件生成函数。

    

#### 103803 索引值异常

 

**错误信息**

 

__RepeatVirtualScrollImpl (eg:1) onCreateNode: for index=(eg:7) with data array length (eg:5), totalCount= (eg:5) out of range error.

 

**错误描述**

 

节点索引超过数据源长度。

 

**可能原因**

 

数据源长度设置有误，或者在不当的时机增删了数据源。

 

**处理步骤**

 

正确设置索引和数据源长度，确保索引不会大于等于数据源长度。

    

#### 103804 Repeat懒加载时非法操作

 

**错误信息**

 

onLazyLoading function executed illegal operation.

 

**错误描述**

 

[懒加载](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-new-rendering-control-repeat#懒加载能力说明)场景下对数据的非法操作。

 

**可能原因**

 

开发者不正确地使用懒加载方法。

 

**处理步骤**

 

合理使用懒加载方法，请参考[数据精准懒加载](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-new-rendering-control-repeat#数据精准懒加载)。

    

#### 103805 默认键值生成失败

 

**错误信息**

 

Repeat(). Default key gen failed. Application Error!

 

**错误描述**

 

[Repeat](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-rendering-control-repeat)的默认[键值](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-new-rendering-control-repeat#键值生成规则)生成失败。

 

**可能原因**

 

开发者错误设置数据源。

 

**处理步骤**

 

合理设置数据源，请参考[键值生成规则](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-new-rendering-control-repeat#键值生成规则)。
