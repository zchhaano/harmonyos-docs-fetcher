# OH_Drawing_Bitmap

 

```
typedef struct OH_Drawing_Bitmap OH_Drawing_Bitmap

```

 

#### 概述

定义为位图，位图是一块内存，内存中包含了描述一张图片的像素数据。

 

**起始版本：** 8

 

**相关模块：** [Drawing](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-drawing)
