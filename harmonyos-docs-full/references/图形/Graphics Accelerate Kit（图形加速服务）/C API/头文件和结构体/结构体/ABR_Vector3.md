## 概述

支持设备PhoneTablet

此结构体描述ABR三维向量。

**起始版本：** 5.0.0(12)

**相关模块：** [GraphicsAccelerate](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/_graphics_accelerate)

## 汇总

支持设备PhoneTablet 

### 成员变量

 支持设备PhoneTablet展开

| 名称 | 描述 |
| --- | --- |
| float x | 三维向量的X轴数据。 取值范围：[0, 360]。 |
| float y | 三维向量的Y轴数据。 取值范围：[0, 360]。 |
| float z | 三维向量的Z轴数据。 取值范围：[0, 360]。 |
