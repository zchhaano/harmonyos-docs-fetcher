## 概述

支持设备PhoneTablet

高精几何重建对象的稠密点云数据。

作为[HMS_AREngine_ARSemanticDense_AcquirePointData](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arengine-capi-arengine#section288942515910)接口输入。

**起始版本：** 6.0.0(20)

**相关模块：** [AR Engine](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arengine-capi-arengine)

## 汇总

支持设备PhoneTablet 

### 成员变量

 支持设备PhoneTablet展开

| 名称 | 描述 |
| --- | --- |
| int32_t id | 当前点的ID。 |
| float x | 当前点的X坐标。 |
| float y | 当前点的Y坐标。 |
| float z | 当前点的Z坐标。 |
| int32_t r | 当前点的颜色，RGBA表示，这里是R的值。 |
| int32_t g | 当前点的颜色，RGBA表示，这里是G的值。 |
| int32_t b | 当前点的颜色，RGBA表示，这里是B的值。 |
| int32_t a | 当前点的颜色，RGBA表示，这里是A的值。 |
| float confidence | 当前点的置信度。 |
